# AccuPrayer — Releases

Accurate Islamic prayer times with athan audio, Qibla compass, smart and customizable notifications, daily prayer counter & tracker, and more.
Built-in app updater to auto-update future releases.

## Download
Go to [Releases](../../releases/latest) to download the latest APK.

### Which APK should I choose?
| File | Device |
| `AccuPrayer-vX.X.X-universal.apk` | Any Android device (recommended) |
| `AccuPrayer-vX.X.X-arm64-v8a.apk` | Modern phones (smaller file) |
| `AccuPrayer-vX.X.X-armeabi-v7a.apk` | Older 32-bit phones |
